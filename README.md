# Zuls Todolist App

     This flexible app maitains multiple lists ( or tasks) per subject. 
     Add subject then Add as many tasks as required for that subject. 
     Add as many subjects as required in order to store all your information. 
     Each subject and task also has priority associated with it.
     Priority is set to 1 but can be edited to be in the range 1 to 5.
     All subjects and tasks are listed in priority order
